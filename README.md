## Setup Instructions (get help as needed): 
1. Fork my 'GEN711_811_Practical-Exam' repo
2. Open vscode. Go to 'file --> new window'. Close all other instances of vscode
3. Hit 'cmd + shift + p' to open the vs code command palette, and start typing 'Remote-SSH:Connect' until you see the 'Remote-SSH:Connect current window to host.'
4. Select RON and enter username and password as needed. 
5. In your home directory at RON, run```git clone https://github.com/YOURUSERNAME/GEN711_811_Practical-Exam.git``` with ```YOURUSERNAME``` replaced with your github username. 
6. In vscode, go to 'file --> load workspace from file', and select the 'exam-env.code-workspace' file in the 'GEN711_811_Practical-Exam' folder. 
7. Make a new branch titled 'My-Exam'. Save, commit, and push to this branch often. To make a new branch, click 'main' on the lower left of vscode.  
8. Paste your notebook/notes into the fileIn vscode (go to 'file --> new file', and save it in the 'GEN711_811_Practical-Exam' repo).For the practical, you should only need one vscode window open, and one terminal open.
### Pro tip: you can right click on either your notes or the exam, and select 'split right' to have both the exam and your notes open at the same time.
